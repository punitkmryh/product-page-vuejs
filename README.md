1. Add to cart
2. Color variant
3. Product in Stock?.
4. Shipping changes.
5. Remove form cart.
6. Users Reviews form with validation.

and using VueJS features like
1. Custom Validation, 
2. Two-way binding,
3. Components features,
4. Computed properties,
5. Event handling,
6. List and Conditional rendering,
